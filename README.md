# Juego Unity — Etapa 1

## Descripción
Proyecto de juego 3D en Unity 2022.3 LTS. Etapa 1: movimiento de personaje y cámara dinámica.

## Controles
| Acción | Tecla |
|--------|-------|
| Moverse | WASD / Flechas |
| Saltar | Espacio |
| Rotar cámara | Mouse |

## Jerarquía de Scripts
- `Assets/Scripts/Player/PlayerMovement.cs` — Movimiento, rotación y estados del jugador (Idle, Run, Jump).

## Requisitos
- Unity 2022.3 LTS
- Cinemachine (Package Manager)
